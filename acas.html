02 · How To Run — Full Workflow
1
ONCE
Setup — clone, install, configure
Run these once when setting up the project for the first time. After this you never need to repeat them.
pip install -r requirements.txt
# 4. Set your Space-Track credentials
cp .env.example .env
# then edit .env: SPACETRACK_USER=you@email.com SPACETRACK_PASS=yourpass
.env.example → .env
requirements.txt
~5 min
2
ONCE
Train the LightGBM model
Must be done before the controller or dashboard can make real predictions. Without this the system falls back to a physics formula. Place your CDM CSV at data_files/satellite_conjunctions.csv first.
# Option A — quick run, no tuning (~3 min on CPU)
python model/train.py
# Option B — with Optuna hyperparameter tuning (RECOMMENDED, ~30 min)
python model/train.py --tune
# Option C — full production run (tuning + 5-fold cross-validation)
python model/train.py --tune --cv
# Batch inference on new data (after training)
python model/train.py --infer --input new_events.csv

TERMINAL 1
Start the orbital physics simulator
The simulator runs at 100× real time — 1 real second = 100 simulated seconds. It writes data_files/satellite_model.json every second. The controller and dashboard read from this file. Keep this running the entire session.
# Keep running in Terminal 1 — do not close
python simulator/orbital.py

4
TERMINAL 2
Start the ACAS autonomous controller
The controller runs a 60-second loop. Each cycle: reads satellite state → fetches TLEs → scans 24h of trajectories → runs LightGBM on every conjunction → decides action. TLE catalog refreshes every 90 minutes. All events are logged to onboard_blackbox.log.
# Keep running in Terminal 2 — do not close
python core/controller.py

04 · Alert Decision Tree
🟢 GREEN
THRESHOLDPc < 1×10⁻⁵
MISS DISTANCEtypically >5 km
ACTIONNo action. Log and continue.
CONTROLLERreturn, sleep until next cycle
DASHBOARDgreen dot on globe
🟡 YELLOW
THRESHOLD1×10⁻⁵ ≤ Pc < 1×10⁻⁴
MISS DISTANCE~1.5 – 5 km
ACTIONMonitor. Alert downlinked to ground.
CONTROLLERlog alert, notify ground station
DASHBOARDyellow badge · increased refresh rate
🟠 ORANGE
THRESHOLD1×10⁻⁴ ≤ Pc < 1×10⁻³
MISS DISTANCE~0.3 – 1.5 km
WITH GROUNDdownlink manoeuvre request, await confirmation
NO GROUND + TCA <2hAUTONOMOUS BURN
NO GROUND + TCA >2hqueue burn, wait for contact
🔴 RED
THRESHOLDPc ≥ 1×10⁻³
MISS DISTANCE<0.3 km (300 m)
WITH GROUNDexecute burn with ground confirmation
NO GROUNDAUTONOMOUS EXECUTION · logged to black box
ΔV SOURCEPPO RL agent → geometric fallback
⚡ Fuel scaling: fuel >50% → standard thresholds · fuel >30% → thresholds ×3 · fuel >15% → thresholds ×8 · fuel <15% → thresholds ×50 (emergency only)  |  ⚡ TLE stale (>48h) → Pc inflated up to 5×  |  ⚡ Battery <20% → Pc sensitivity ×1.5
